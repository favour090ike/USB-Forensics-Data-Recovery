# USB-Forensics-Data-Recovery
The goal of this project was to perform a digital forensic investigation on a USB flash drive (WINSETUP (D:)) that had been reset. The primary focus was to demonstrate how forensic tools like Autopsy can recover data from unallocated space even after a drive has been wiped or reformatted.
TOOLS USED 
 1. Autopsy
 2. PhotoRec Carver
