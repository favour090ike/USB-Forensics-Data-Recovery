##  Project Overview
This project demonstrates a basic digital forensic investigation on a USB flash drive. The goal is to analyze a storage device, recover deleted files, and document forensic findings using Autopsy.

The project simulates a real-world forensic scenario where an investigator examines removable media for evidence.

##  Objectives
- Perform a digital forensic investigation on a USB drive
- Recover deleted files and analyze metadata
- Understand forensic workflow and evidence handling
- Practice proper forensic documentation
